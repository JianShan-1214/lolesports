# 產品概述

## LOL 比賽通知系統

一個最小可行產品 (MVP) 應用程式，讓使用者能夠訂閱特定的英雄聯盟戰隊，並在這些戰隊有比賽時透過 Telegram 接收通知。

### 核心功能
- 透過網頁介面管理戰隊訂閱
- 從 Leaguepedia API 獲取即時比賽資料
- 為訂閱戰隊自動發送 Telegram 通知
- 使用者友善的訂閱管理介面

### 目標使用者
- 英雄聯盟電競愛好者
- 想要隨時掌握喜愛戰隊比賽動態的粉絲
- 偏好自動通知而非手動查看的使用者

### 核心價值主張
透過自動化、及時的通知直接傳送到您的 Telegram，絕不錯過您喜愛的 LoL 戰隊比賽。
